Símbolos, Sintaxis, Esqueleto del programa

La sintaxis de este lenguaje es algo diferente debido a la forma de su escritura. Siendo que las cadenas de texto se escriben entre comillas dobles. Es posible escribir varias expresiones matemáticas directamente en ghci y obtener un resultado. Prelude es el prompt por defecto de GHCi.Las funciones se llaman colocando los argumentos directamente después del nombre de la función. No hay paréntesis en la llamada a la función.

Las funciones putStr y putStrLn imprimen cadenas en la terminal. La función print imprime cualquier tipo de valor. (Si se hace print de una cadena, tendrá comillas). Si necesita múltiples acciones I/O en una expresión, puede usar un bloque do. Las acciones se separan con punto y coma. Si desea declarar el tipo de variable, solo es necesario utilizar “::” y escribir seguido a esto el tipo de dato a ingresar. En este lenguaje de prgramación no es necesario un esqueleto, por lo que simplemente se empieza a escribir el codigo automaticamente, pero respetando las reglas de sintaxis con las que cuenta que fueron planteadas anteriormente.

Aplicación para descargar

En este lenguaje de programación se cienta con plataformas que ayudan a correr el programa que fue escrito en el archivo fuente, algunas plataformas son:

Glasgow Haskell Compiler (GHC). Siendo el mas utilizado por los programadores que utilizan este lenguaje de programación para sus trabajos.

Otra herramienta que puede ser utilizada es el Typing Haskell in Haskell.

Paradigma

Haskell es un lenguaje funcional (donde todo se hace con llamadas a funciones), estático, implícitamente tipado (los tipos los revisa el compilador, pero no hace falta declararlos), perezoso (nada se hace hasta que es completamente necesario). Los lenguajes populares más parecidos son la familia ML.

Ejemplo

dadas dos listas de enteros de igual longitud, devuelve una lista de pares (min,max), donde min y max son el mínimo y el máximo entre los elementos de ambas listas en la misma posición.

zipSort :: [Integer] -> [Integer] -> [(Integer,Integer)]

zipSort [] [] = []

zipSort (x:xs) (y:ys) | x>y = (x,y) : zipSort xs ys

| otherwise = (y,x) : zipSort xs ys
